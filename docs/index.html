By design, snaptailor is read-only. It never executes MCP commands, scripts, or plugins. It never phones home. It never alters your configuration. It stores metadata-first and omits raw secrets. The integrity of your workshop is our first principle.

Services of the Workshop

Every tool a master tailor needs to preserve the perfect fit.

Snapshot

Preserve the complete state of your agent — prompts, MCP servers, skills, tools, and settings — in a single point-in-time capture.

Semantic Diff

Pinpoint every change between any two snapshots. See prompt edits, MCP additions, permission shifts, and more.

Security Audit

Spot risky alterations: executable config additions, remote MCP servers, permission wildcards, and reproducibility gaps.

Provenance

Trace every setting to its origin — user config, project config, environment variables, or MCP servers.

Restore Planning

Generate a non-mutating restore plan. Review every change before committing. Roll back with confidence. v0.2

Export & Share

Export redacted reports and bundles. Share your perfect fit with the team. v0.2

How to Use

Four steps to a perfectly fitted agent environment.

01

Measure

Scan your project to detect agents, MCP servers, skills, and settings.

snaptailor scan --project . --explain
02

Cut

Create a baseline snapshot to preserve the current configuration.

snaptailor snapshot create --name baseline --project .
03

Fit

After changes, diff against baseline and audit for risks.

snaptailor diff baseline current --project .
snaptailor audit current --project .
04

Restore

Generate a restore plan and roll back to the perfect fit.

snaptailor restore --snapshot baseline --dry-run --project .
