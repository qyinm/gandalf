# Bundle Format

Gandalf uses `.gandalf` bundles as portable profile export files.

## Structure

A `.gandalf` file is a gzipped tar archive containing:

- `manifest.json` — metadata about the bundle
- `snapshots/` — snapshot data
- `profiles/` — profile configurations

## Usage

```bash
gandalf bundle export --name baseline --out backup.gandalf --project .
gandalf bundle inspect backup.gandalf
gandalf bundle import backup.gandalf --dry-run --project .
```

## Security

Bundles do not contain raw secret values. Environment key names are preserved but values are redacted.
