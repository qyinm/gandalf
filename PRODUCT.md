**snaptailor** 제품 설명

### 한 줄 요약
**AI 에이전트를 완벽하게 재단하고, 그 상태를 스냅샷처럼 저장·관리·복원하는 디지털 양장점**

### 상세 설명
snaptailor는 AI 에이전트 개발자와 파워유저를 위한 **에이전트 환경 관리 도구**입니다.

단순한 프롬프트 하나만 저장하는 것이 아니라, **실제 사용 중인 에이전트의 완전한 상태 전체**를 한 번에 스냅샷으로 저장하고 언제든지 정확하게 복원할 수 있게 해줍니다.

### 관리하는 대상
- System Prompt & Role 설정
- Tools / Functions / MCP Servers
- Memory & Vector Store 설정
- Workflow / Multi-Agent 구조
- Model (LLM) 및 Reasoning Config
- 설치된 커스텀 에이전트 (oh-my-openagent, Ouroboros 등)
- Skills, Plugins, Extensions 폴더 전체
- .env 파일, 로컬 config, hooks 등

### 핵심 기능
- **Snapshot** — 현재 에이전트 환경을 한 번에 저장 (Git Snapshot + Docker Image 같은 느낌)
- **Load / Restore** — 원하는 스냅샷으로 순식간에 환경 복원
- **Diff** — 두 스냅샷 간 변경점 상세 비교 (프롬프트, 스킬, 툴 변경 사항 강조)
- **Scan** — 시스템에 설치된 다양한 AI 에이전트와 설정 자동 탐지
- **Export / Share** — 스냅샷을 번들 파일로 만들어 팀원에게 공유
- **Versioning** — 에이전트의 버전 관리 (v1.2-research, v2.1-productivity 등)

### 컨셉
영국 Savile Row의 고급 양장점처럼,  
**“당신의 AI 에이전트를 최고의 상태로 맞춤 재단한다”**는 철학을 가지고 있습니다.

### 사용 예시
- 연구용 에이전트 → 생산용 에이전트로 빠르게 전환
- 실험한 설정이 마음에 들면 스냅샷으로 안전하게 보관
- 팀원에게 “이 에이전트 설정 그대로 써봐”라고 공유
- 에이전트가 망가졌을 때 이전 좋은 상태로 10초 만에 롤백
